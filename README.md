# HR Employee Attrition Dashboard

## 📊 Project Overview
Built an interactive HR analytics dashboard to analyze employee attrition, salary distribution, and department‑wise workforce trends.  
Goal: Help HR teams identify key factors driving attrition and improve retention strategies.

## 🛠️ Tech Stack
- Power BI (dashboard creation, interactive visuals)
- Excel (data cleaning, pivot tables)
- Python (optional preprocessing, if used)

## 🔍 Steps Taken
1. Cleaned HR dataset (handled missing values, standardized department names).  
2. Created measures and KPIs in Power BI:
   - Attrition rate by department  
   - Average salary by role  
   - Employee count by age group and gender  
3. Designed interactive visuals:
   - Attrition by department (bar chart)  
   - Salary distribution (histogram)  
   - Employee demographics (pie chart)  
4. Published dashboard for stakeholder review.

## 🚀 Key Insights
- Overall attrition rate ≈ **15%**.  
- **Sales department** showed the highest attrition compared to other functions.  
- Average salary differences across departments highlighted retention challenges.  
- Younger employees (<30 years) had higher turnover compared to senior staff.

## 📂 Files in this Repository

- `HR_Attrition_Data.xlsx` – cleaned dataset

## ▶️ How to Use
1. Open `HR_Attrition_Dashboard.pbix` in Power BI Desktop.  
2. Explore interactive visuals (filters by department, age, salary).  
3. Use insights to support HR decision‑making.
